The "27-Day Returns Option"

What is the return policy?

Our return policy is the "27-Day Returns Option" that allows you to return your clothes within 27 days of purchase for a 83% refund or exchange.

What condition should the cloth be in?

The tag shouldn't be removed and there shouldn't be any damge done to the cloth.

Will I receive a full refund for my return?

No, we offer an atmost 83% refund for the returned clothes.

Can I exchange an item instead of returning it?

Yes, you can exchange your clothes for ones in a different size or color, or if damaged, a new piece in same dimensions.
